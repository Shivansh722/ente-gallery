# Ente Gallery (Level 1)

A Flutter photo gallery app displaying a shuffled grid of 60 images.

## Setup

1. Ensure you have 60 images named `photo_1.jpg` through `photo_60.jpg` in `assets/gallery/`.
2. Install dependencies:
   ```sh
   flutter pub get
   ```
3. Run the app:
   ```sh
   flutter run
   ```

## Features

- System-adaptive theme (light + dark mode)
- 3-column grid of shuffled images
- Images randomize on each app launch
- Clean, readable code with minimal state management
